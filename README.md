# TTSWeb

Full-stack Text-to-Speech web application leveraging Qwen3-TTS.

## Project Structure

- **[backend/](./backend)**: FastAPI backend service providing TTS generation, voice cloning, and tokenizer APIs.
- **[frontend/](./frontend)**: React + Vite + TypeScript frontend with a modern UI for generating and managing speech.

## Getting Started

Refer to the individual service directories for specific setup instructions:
- [Backend Setup](./backend/README.md)
- [Frontend Setup](./frontend/README.md)

## Features

- **Custom Voice Generation**: High-quality TTS using Qwen3-TTS.
- **Voice Design & Cloning**: Create and use custom voice profiles.
- **Modern UI**: Dark mode, interactive audio visualizer, and streamlined generation flow.
- **Real-time Streaming**: Efficient audio delivery via WebSockets.
